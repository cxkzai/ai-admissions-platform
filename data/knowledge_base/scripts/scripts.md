## 首次接待话术（开场）

### 模板 A：家长主动咨询
```
您好，我是码力星编程的 {{顾问姓名}}，请问是 {{家长称呼}} 吗？
感谢您关注我们～ 我先简单介绍一下：
我们专注 8-15 岁少儿编程，在成都有 3 个校区，师资都是全职，平均教龄 4 年。
您家孩子是 {{孩子年龄}} 岁对吧？方便说一下孩子的基本情况吗？比如：
  1. 之前有没有接触过编程？
  2. 主要是想培养兴趣，还是有竞赛/升学方面的考虑？
  3. 您和孩子倾向线上还是线下？
```

### 模板 B：老学员推荐
```
您好～ 您是 {{老学员姓名}} 妈妈推荐过来的对吧？
{{老学员姓名}} 在我们这里学了 {{时长}}，进步非常大，孩子特别棒！
您家孩子是 {{孩子年龄}} 岁？我们可以先安排一节免费试听，让老师评估一下孩子的水平，再推荐合适的课程。
```

## 挖需 5 问

```
好的，我已经了解了基本情况。为了更精准地推荐课程，我再问几个问题：
1. 【年龄】孩子现在几年级？生日是哪个月的？
2. 【兴趣】孩子平时喜欢玩什么？游戏 / 动画 / 拼积木？
3. 【基础】之前有没有接触过编程？在家玩过 Scratch 吗？
4. 【目标】家长您希望孩子学完能达到什么目标？
   - A. 兴趣启蒙、培养思维
   - B. 参加比赛、升学加分
   - C. 留学作品集准备
5. 【预算】您的预算大概在多少？
   - A. 5000-8000 元/年
   - B. 8000-15000 元/年
   - C. 15000 元以上 / 不设限
```

## 异议处理

### "太贵了"
```
理解您的顾虑。咱们的价格确实比直播大班课贵一些，但咱们是 6 人小班 + 全职老师 + 线下校区，成本确实高一些。

不过您可以算一笔账：
- 我们的课单价约 ¥{{单课时价格}}（共 {{总课时}} 课时）
- 平均到每周约 ¥{{周均价格}}
- 孩子学完能独立做出完整作品

而且现在报名享限时优惠 + 首课 9 折，能省 {{优惠金额}}。

您也可以先给孩子报一节 ¥{{试听课价格}} 的试听课看看效果，再决定也不迟。
```

### "再考虑考虑"
```
好的，您考虑考虑是应该的，毕竟是给孩子报班。
不过我想了解一下您主要在考虑哪些方面？是价格、效果、还是时间？
我们可以针对性地解决您的顾虑。
另外我们的优惠活动每月 1-15 日开放，如果您想报名，建议这周内决定，能省 {{优惠金额}}。
```

### "和别家对比"
```
理解，您对比一下是应该的，毕竟是长期投入。
我想问下您对比的是哪家？我们和 {{竞品}} 的主要区别是：
1. **教学形式**：我们是 6 人小班 + 线下，{{竞品}} 是大班直播
2. **师资稳定性**：我们全职老师平均教龄 4 年，{{竞品}} 的老师 1 年流失率 40%+
3. **竞赛成绩**：我们培养了 50+ NOIP 一等奖学员
4. **价格**：我们小班课比一对一便宜，比直播大班贵，但性价比最优

我可以给您发一些详细的对比资料，您看完再做决定也不迟。
```

### "孩子没基础"
```
完全不用担心！我们 Scratch 启蒙班就是为零基础孩子设计的：
- 图形化拖拽编程，不需要英文基础
- 8 岁孩子就能学会
- 每节课完成一个小作品，孩子越学越有兴趣
- 课后有家长群反馈，老师会同步孩子的进步

我们班上有不少完全零基础的孩子，3 个月后都能独立做出完整游戏。
要不先给孩子安排一节免费试听，看孩子的反应再决定？
```

## 试听邀约话术（3 套）

### 套 A：限时优惠驱动
```
{{家长称呼}}，我们本月有一个限时活动：
- 试听课当天报名立减 ¥{{优惠金额}}
- 老学员推荐双方各得 ¥500 抵扣
- 每月 1-15 日开放，下个月就涨价了

要不这周六 / 周日带孩子来校区试听一下？我们的试听课是完全免费的，孩子上完课喜欢的话，当天就能锁定优惠。
```

### 套 B：科学评估驱动
```
{{家长称呼}}，光听我介绍您可能感受不到孩子的水平差异。
我建议您带孩子来校区做一次"编程能力评估"：
- 我们的明星老师 {{老师姓名}} 会亲自评估
- 60 分钟 1v1 测出孩子的逻辑思维和编程天赋
- 评估完后老师会给出个性化的学习路径建议
- 完全免费，不报名也没压力

您看本周六 / 周日上午方便吗？
```

### 套 C：成果展示驱动
```
{{家长称呼}}，我们最近有一位和您家孩子情况很像的学员：
{{学员故事简述}}
他现在学了 6 个月，已经能独立做出完整作品。

您家孩子如果有兴趣，我可以把孩子的"目标作品"提前告诉您。
让孩子试听课时就有目标感，效果会更好。

本周六 / 周日下午您方便带孩子来校区吗？
```

## 试听后跟进话术

### 24h 内（首跟）
```
{{家长称呼}}，您好～ 昨天试听课您觉得怎么样？
{{孩子姓名}} 表现 {{具体表现}}，老师反馈他 {{具体亮点}}。

我们这周还有 {{优惠活动信息}}，如果您想给孩子报名，建议这周内决定，能锁定优惠价 ¥{{优惠价}}。

有任何问题随时问我～
```

### 3 天后（深度跟）
```
{{家长称呼}}，您好～ 上次试听已经 3 天了，您考虑得怎么样？
我整理了一份资料给您：

期待您的回复～
```

```
{{家长称呼}}，您好～ 我们的本月优惠活动本周日截止，明天是最后一个工作日。

优惠结束后恢复原价 ¥{{原价}}，相当于多花 {{差额}}。

我等您回复～
```
