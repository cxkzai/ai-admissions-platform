## Scratch 编程启蒙

**适合年龄**：8-10 岁
**课时**：48 课时（每课时 90 分钟，约 4-6 个月）
**班型**：6 人小班
**原价**：¥6,800
**限时优惠**：¥5,800（首课优惠 + 老学员推荐）

### 课程目标
从零开始培养孩子的编程思维与逻辑能力，无需任何英文基础。孩子将学会用 Scratch 创作动画、游戏、互动故事，理解顺序、循环、条件判断等基础编程概念。

### 课程亮点
- **零门槛入门**：图形化拖拽式编程，不需要认识英文单词
- **游戏化教学**：每节课完成一个有趣的小项目（迷宫游戏、接水果、飞机大战）
- **作品可展示**：每阶段都有可分享的作品，家长能在小程序看到孩子作品
- **逻辑思维启蒙**：培养问题拆解、模式识别、抽象思维等核心能力

### 课程内容
- 阶段一：Scratch 基础（16 课时）—— 角色、舞台、事件、运动、声音
- 阶段二：Scratch 进阶（16 课时）—— 变量、条件、循环、克隆、广播
- 阶段三：项目实战（16 课时）—— 完成 3 个完整作品（迷宫 / 消除游戏 / 互动故事）

### 师资配置
王晓明老师（6 年少儿编程教学经验）+ 陈思雨老师（亲和力强）

### 适合学员
- 8-10 岁，零基础或简单接触过 Scratch
- 家长希望培养逻辑思维、专注力
- 对游戏/动画有兴趣的孩子

## Python 编程基础

**适合年龄**：10-12 岁
**课时**：60 课时（约 5-7 个月）
**班型**：6 人小班
**原价**：¥8,800
**限时优惠**：¥7,800

### 课程目标
从图形化编程过渡到代码编程，掌握 Python 基础语法与实战项目。能独立开发小游戏、数据处理脚本、Web 爬虫入门，为后续学习 C++ 算法和信奥竞赛打下基础。

### 课程亮点
- **平滑过渡**：从 Scratch 思维迁移到 Python 代码，配有专门的思维转换训练
- **实战项目丰富**：5 个完整实战项目（猜数字、贪吃蛇、数据图表、聊天机器人、简单爬虫）
- **为竞赛铺垫**：课程内容覆盖 CSP-J 入门级知识点
- **英文能力同步提升**：常用编程英文词汇自然习得

### 课程内容
- 阶段一：Python 基础语法（20 课时）—— 变量、数据类型、运算符、输入输出
- 阶段二：控制流与函数（16 课时）—— 条件、循环、函数、模块
- 阶段三：数据结构入门（12 课时）—— 列表、字典、字符串操作
- 阶段四：实战项目（12 课时）—— 5 个完整项目开发

### 师资配置
李梦琪老师（清华计算机硕士）+ 王晓明老师

### 适合学员
- 10-12 岁，有 Scratch 基础或逻辑思维较好
- 准备参加 CSP-J 比赛
- 家长希望孩子掌握真正的代码编程能力

## NOIP 信奥冲刺

**适合年龄**：13-15 岁
**课时**：100 课时（集训 7-8 个月）
**班型**：4-6 人精品小班
**原价**：¥16,800

### 课程目标
冲击 NOIP（现 CSP-S）省级一等奖及以上奖项，为高考强基计划、综合评价、自主招生铺路。

### 课程亮点
- **历年获奖率 60%+**：近 3 年 60% 学员获 CSP-S 一等/二等
- **个性化训练计划**：每个学员独立培养方案
- **模拟赛体系**：每月 2 次全真模拟赛 + 详细讲评
- **强基计划衔接**：配套强基计划面试辅导

### 课程内容
- 高级算法（32 课时）—— 图论、数论、字符串
- 高级数据结构（24 课时）—— 线段树、平衡树、并查集
- 动态规划进阶（20 课时）—— 状态压缩、概率 DP
- 真题精讲 + 模拟赛（24 课时）

### 师资配置
张子豪老师 + 周建国老师（退役 ACM 队员）

### 适合学员
- 已完成 C++ 算法进阶
- 数学竞赛或编程竞赛基础扎实
- 目标：NOIP 获奖、强基计划、综合评价
